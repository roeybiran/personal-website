## User Guide

### Item Actions

| Keyboard Shortcut | Selected Item | Action                               |
| ----------------- | ------------- | ------------------------------------ |
| ⇧⌘Q               | Running App   | Quits the app                        |
| ⇧⌘H               | Running App   | Hides or unhides the app             |
| ⇧⌘W               | Window        | Closes the window                    |
| ⇧⌘M               | Window        | Minimizes or un-minimizes the window |
